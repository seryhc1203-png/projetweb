L'objectif de ce projet est de concevoir une application en Java intégrant trois approches d'intelligence artificielle : A*, MinMax et un perceptron. Le projet met l'accent sur une conception orientée objet, notamment la séparation entre classes de données, classes de traitement et interface graphique.

Référence : Sujet AI.

Objectifs

  • Modéliser les données nécessaires au fonctionnement du logiciel (diagramme de classes).
  • Mettre en place des environnements de simulation (labyrinthe, plateau de jeu, parterre de roses).
  • Implémenter les algorithmes A*, MinMax et un perceptron en Java.
  • Développer une IHM graphique permettant de visualiser et d'animer les simulations.

Équipe

Membres : HAMICI Serina, ADOUNSINA Melvina, DIABY Moustapha.

Information projet

  • Langage : Java (moteur + classes de données)
  • IA : A*, MinMax, Perceptron
  • Objectif : simulation + visualisation
